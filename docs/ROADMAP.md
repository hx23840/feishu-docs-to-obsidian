# Roadmap

## Near Term

- Add fixture-based converter tests.
- Improve table conversion.
- Add import progress for large documents.
- Add safer conflict handling on refresh.
- Support custom attachment subfolders per document.

## Later

- Direct Feishu Open API mode without `lark-cli`.
- Batch import.
- Better embedded file handling.
- Wiki tree import.
- Optional bidirectional sync metadata.

## Non-goals for the MVP

- Mobile support.
- Bypassing Feishu permissions.
- Storing Feishu access tokens in the plugin.
